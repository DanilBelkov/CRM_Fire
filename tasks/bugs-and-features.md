# Задачи: Баги и Новые функции

## Баги

- [x] **Дублирование фото в галерее при сохранении в карточке**
  - Исправлено: `_newGalleryPhotos.Clear()` перенесён до `RefreshGallery()`.
  - Причина: `RefreshGallery()` конкатенировал `_newGalleryPhotos` с уже сохранёнными в БД фото, создавая дубли.

- [x] **Исчезает значение в поле "Ответственный" при сохранении в карточке**
  - Исправлено: добавлен метод `SelectResponsiblePerson()`, который ищет `DirectoryUser` с совпадающим `FullName` и устанавливает `SelectedItem`.
  - Причина: `ComboBox.Text` не синхронизируется с `ItemsSource` из `DirectoryUser`.

  - В режиме просмотра: кнопки "ОК" и "Редактировать".
  - В режиме редактирования: кнопки "Сохранить" и "Отмена".
